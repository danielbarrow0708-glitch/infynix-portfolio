## Getting Started
Use node version 18.15.0 
To run this React project locally, follow these steps:

1. Clone this repository to your local machine:
   git clone https://github.com/Idea-Maker-Agency/chatgpt-scaffold-frontend

2. Navigate to the project directory:
   cd  chatgpt-scaffold-frontend

3. Install dependencies using npm or yarn:
   npm install
# or
yarn

4. Start the development server:
npm run dev
# or
yarn dev
5. Open your browser and visit http://localhost3000 to see the application.

Chat can be managed through following env whose value will be true or false
VITE_CHAT=true
Base Url of api is managed by following env variable 
VITE_API_URL="https://chatgpt.ideamakr.com/backend/api/v1"
